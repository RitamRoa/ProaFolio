# Ritam Roa
  Portfolio website in a very minimalistic way (updated)

### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/RitamRoa/ProaFolio.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Run the development server:
    ```bash
    npm run dev
    ```
